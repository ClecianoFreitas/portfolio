Tecnologias

HTML

HTML (HyperText Markup Language) é uma linguagem de marcação usada na criação e estruturação de páginas web, utilizando tags para definir e organizar elementos.

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para definir o layout e a apresentação visual de documentos HTML em páginas web.

JavaScript

JavaScript é uma linguagem de programação amplamente utilizada para criar interatividade em páginas web, possibilitando a manipulação dinâmica do conteúdo..

GitHub

O GitHub é uma plataforma de hospedagem de código-fonte, facilitando o controle de versão e colaboração em projetos de desenvolvimento de software.
